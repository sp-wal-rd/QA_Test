▸ HOW TO CONNECT YOUR GOOGLE SHEET
  1. Open a Google Sheet → Extensions → Apps Script
  2. Paste this code:

    function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); var row = [ new Date(), data.heading, data.tester, data.build, data.notes, data.checkedCount, data.summary, data.customItems ]; sheet.appendRow(row); return ContentService.createTextOutput( JSON.stringify({status:"ok"}) ).setMimeType(ContentService.MimeType.JSON); }
  3. Deploy → New Deployment → Web App → "Anyone" → Copy URL
  4. Paste the URL in the config bar at the top of this page
